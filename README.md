Welcome to beer cards! 

This app will allow users to rate their favorite (and not so favorite) beers and share them with their friends.

This app is still in early stages of development. 



The app currently has a "create" page where users can add a new beer into the database: 

![ecf99269bc7e4deed72c4e42a9b411ee](https://user-images.githubusercontent.com/73203671/152656763-15722ecf-0756-4db7-8605-cf27f02dea90.png)



Users will fill out the form and once hitting submit their beer will be added to the beer collection and will be able to be viewed by all users. 
![ae632fc3db27c5a961118ef698d66f77](https://user-images.githubusercontent.com/73203671/152656918-3de42ab4-92db-4527-a04b-b31cb906afc1.png)




The beer collection page will look something like this with many differnt types of beers. 
![b3d927cb9e6121c3c6fda96f2a64c876](https://user-images.githubusercontent.com/73203671/152656764-aade9af3-d22c-4852-bcf7-bdb32e373960.png)



The app currently supports an edit and delete feature where the posts can be edited or deleted from the database. 

The goal is to have a grid like display of all the beer cards on the main page and a sort function where users can sort through different types of beer to see the highest rated ones. 

The app currently does not support user account creation and that will be the next step.
